# ConsoleWarGame

Невелика консольна гра на C# (.NET), де гравець проходить 3 рівні бою проти випадкових ворогів.

## Що є в грі

- Введення імені гравця на старті.
- Покроковий бій: натискаєш `Enter` для кожної атаки.
- Випадкова шкода в межах `MinAttack-MaxAttack`.
- Бонуси, які гравець обирає під час гри:
  - `+20%` до атаки,
  - `+25` фіксованої шкоди,
  - `Critical strike` (`17%` шанс, `+50%` до поточного удару).
- Кожен бонус можна взяти лише 1 раз за гру.
- Ліміт бонусів за рівень:
  - Level 1: до 1 бонусу,
  - Level 2: до 2 бонусів,
  - Level 3: до 3 бонусів.
- На Level 3 ворог має подвоєне HP.
- Після перемоги/поразки результат записується в `game_results.txt`.

## Структура проєкту

- `Program.cs` — головний цикл гри, логіка рівнів і бонусів.
- `Character.cs` — базовий абстрактний клас бійця.
- `PlayerCharacter.cs` — логіка гравця (бонуси, крит).
- `Opponent.cs` — логіка ворога.
- `CombatStats.cs` — структура стартових характеристик.
- `EnemyType.cs` — enum типів ворогів.
- `IDamageDealer.cs` — інтерфейс нанесення шкоди.

## Запуск

```bash
dotnet run
```

## Збірка

```bash
dotnet build
```
